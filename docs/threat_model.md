# Threat Model (Summary)

## Assets
- Group integrity and moderation actions
- User trust and privacy

## Threats
- Spam links and scam content
- Flooding and repeated content
- Evasion using unicode tricks

## Mitigations
- Link and keyword detection
- Rate and repetition heuristics
- Minimal data retention

## Risks
- False positives on legitimate links
- Bot losing admin permissions
